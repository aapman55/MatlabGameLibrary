# Lights on look up tables

## 3x3 look up table
These add up. If you have 1 and 2 then press the corresponding top lights for both 1 and 2

| Left on bottom row | Push on top row |
|:-------------------|----------------:|
| 1                  | 1, 2            |
| 2                  | 1, 2 ,3         |
| 3                  | 2, 3            |

## 4x4 look up table
No need. Always solves itself.

## 5x5 look up table


| Left on bottom row | Push on top row |
|:-------------------|----------------:|
| 1, 2, 3            |               2 |
| 1, 2, 4, 5         |               3 |
| 1, 3, 4            |               5 |
| 1, 5               |            1, 2 |
| 2, 3, 5            |               1 |
| 2, 4               |            1, 4 |
| 3, 4, 5            |               4 |
